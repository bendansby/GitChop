Changes

  • Open Repo… no longer leaves an empty window behind if you cancel the picker. The window only opens once you've actually chosen a repo.
  • "Close Tab" is now "Close Repo" in the File menu, matching the in-app language.
  • System window tabbing disabled. GitChop's own tab strip is the multi-repo story, so macOS's Show/Hide Tab Bar and Merge All Windows items no longer appear in the Window menu.
