Initial preview

  • Visual git rebase -i. Open a repo, see the last N commits as a list, drag to reorder, change verbs with one click. Apply runs git rebase behind the scenes.
  • Verbs in v0.1: pick, squash, fixup, drop. Reword and split-commit (edit) come in v0.2.
  • Auto-backup ref. Every Apply writes refs/gitchop-backup/<timestamp> first; if anything goes wrong GitChop aborts and resets HEAD back to it.
  • Diff pane. Click any commit to see its git show in the right pane.

Known limits

  • No reword — change subjects via terminal for now (or wait for v0.2).
  • No conflict resolver — if the rebase hits a conflict, GitChop aborts and rolls back. Resolve conflicts in your terminal/mergetool, then come back.
  • Loads the most recent 12 commits. Custom range picker is v0.2.
